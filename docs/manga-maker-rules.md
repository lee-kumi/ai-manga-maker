# AI漫画メーカー 基本思想＋制作ルール

この文書は、AI漫画メーカーv2（`v2.html`）に組み込まれている思想とルールを、人間が読める形でまとめたものです。
アプリを改修するとき・「このアプリはルールを満たしているか？」を照合するとき・別のAIに相談するときの基準にします。

- 対象アプリ：https://lee-kumi.github.io/ai-manga-maker/v2.html
- 抽出元：`v2.html`（2026-07-07時点のプロンプト生成コード）
- 姉妹アプリ：Cover Compass（`cover-maker.html`）のルールは本文書末尾の「関連」を参照

---

## 基本思想

**漫画は「絵」ではなく「物語」である。**

- 目的は、絵のうまさを見せることではなく、**読者の心を動かし、想いを伝えること**
- AI漫画づくりで一番失敗しやすいのは、**企画（STEP1）を雑にして、いきなり画像生成から始めること**。テーマとキャラが固まっていないと、ページごとに顔が変わり、雰囲気がバラバラになる
- だから必ず **企画 → キャラ・画風 → コマ割り → 画像生成** の順に進む。画像生成は最後
- このツール単体で漫画を完成させるのではない。**画像生成AI（ナノバナナ・ChatGPT）に渡すプロンプトを設計する場所**である
- **顔ブレ防止が品質の生命線**。キャラクターシートを必ず作り、画像生成のたびに参照画像として添付する

---

## 全体ルール

| # | ルール |
|---|---|
| 1 | 4つのSTEPを順番に進める（飛ばさない） |
| 2 | 各STEPの入力内容は次のSTEPのプロンプトに引き継がれる。「次のSTEPへ」ボタンで進む |
| 3 | 生成したプロンプト・AIの出力は、作品専用のGoogleドキュメント（作業用ドキュメント）に都度保存する。これが制作全体の設計図になる |
| 4 | データはブラウザのlocalStorageのみに保存される。大事な作品はバックアップ機能でファイル保存する |
| 5 | キャンバスサイズは **1000×1414px（A4縦）** を基準とする（Canva編集がしやすいサイズ） |

## STEP1：企画書のルール

### ストーリー設計で特に大事な5つ（必須）

1. **悩み**（出発点）— 主人公が最初にどんな状態で、何に悩んでいるか
2. **本音**（心の声）— 飾らない正直な気持ち。読者が共感するのはきれいな言葉より正直な気持ち
3. **壁**（挫折）— 途中でぶつかる問題。壁があることで物語に深みが出る
4. **カギ**（気づき）— 前に進むための気づき。何を学んで変わったのか
5. **結末** — 最後にどう変わるのか＋読者に何を届けたいのか

任意項目：事件（転換点）・転機・成長。入れるほど物語が具体的になる。

### ペルソナのルール

- この漫画を一番読んでほしい**たった一人**に届けるつもりで書く
- 読者の悩みや気持ちまで書くと、AIが読者に寄り添ったストーリーを作りやすい

### タイトルの条件（優先順位順）

1. **【最優先】検索キーワードを含める** — 読者がAmazon・SNS・Googleで実際に検索する言葉を入れる。気の利いた抽象的なタイトルはNG。無名の著者は検索性ファースト（例：「AI漫画」「50代から」「初心者でも」「〇日で」）
2. **【次点】ベネフィットを明確に伝える** — 読むと何が得られるかをタイトルに直接入れる（例：「〇日でできる」「素人でも作れる」）
3. **【補強】数字を入れて信頼感を上げる**（7日・3ステップ・90日など）
4. 主人公の悩みや変化もタイトルから伝わると尚良い

## STEP2：キャラ・画風のルール

### 画風

- 画風は**全コマ共通で1つに固定**する（STEP3・4のプロンプトに自動反映）
- ボタン選択か自由入力の**どちらか一方**を使う（自由入力に書くとボタン選択は自動解除）
- 自由入力は、理想の画像のプロンプトをAIに出してもらって貼り付ける使い方が有効

### キャラクターシート（顔ブレ防止の要）

構成は7点セット＋ラベル英語表記：

1. 全身正面（Front View）
2. 全身側面（Side View）
3. 全身背面（Back View）
4. 顔アップ（Face Close-up）
5. 表情集（Expressions）：通常・笑顔・怒り・悲しみ・驚き・困り顔
6. 小物設定（Props）
7. カラーパレット（Color Palette）：髪色・肌色・服装・アクセントカラー

レイアウト：白背景・資料集形式・全身が切れない配置・高解像度。

### 背景なし素材画像のルール

- 背景：**真っ白（#FFFFFF）のみ**。背景要素・影・グラデーション・床・地面を一切描かない
- 輪郭をはっきり描く・全身が収まる配置・高解像度
- 透過が必要なら remove.bg や Canvaの背景除去で処理

## STEP3：コマ割りシナリオのルール

### セリフのルール

- **句点（。）は絶対に使わない**（漫画のルール）
- 1コマのセリフは**30文字以内**
- 「セリフ」「モノローグ（心の声）」「ナレーション（地の文）」は必ず分ける

### コマ割りのルール

- スマホ読み前提：**1ページで伝えることは1つ**に絞る
- クライマックス・感情ピークは**大ゴマ**（高さ40〜60%）で止める
- 構図を毎ページ変える（正面・俯瞰・煽り・クローズアップ・引きを混ぜる）
- 最後のコマは**「めくり」**＝全部見せず、次が読みたくなる所で止める
- 1ページあたりのコマ数目安：3〜4コマ（クライマックスは1〜2コマ・テンポシーンは4〜5コマも可）

### レイアウトパターン（ページ構成表用・毎ページ変える）

同じパターンを**3ページ以上続けない**こと。

| パターン | 内容 |
|---|---|
| A | 横割り（2〜4コマ縦並び） |
| B | 上大コマ＋下縦割り2コマ |
| C | 上縦割り2コマ＋下大コマ |
| D | 縦割り2コマ（左右均等） |
| E | 大ゴマ1コマ（クライマックス・見開き風） |
| F | 小コマ4〜5コマ（テンポ重視） |

※AIへの出力形式指定：表形式（| --- |）は使わせない。テキスト形式で出力させる。

## STEP4：画像生成のルール

### ⚠️ 最重要ルール（2つ）

1. **コマの読む順序は必ず「右上→左上→右下→左下」（日本式：右から左）**。左から右に読む配置は絶対禁止
2. **添付した参照画像のキャラクターの外見（顔・髪型・服装・小物含むすべて）を全コマで忠実に再現**。参照画像と異なる見た目は禁止

### キャンバス・基本仕様

- サイズ：1000×1414px（A4縦）
- 読み方向：右から左（日本式）
- コマ枠：はっきり描く・黒枠。コマ間に白い余白（ガター・目安10〜15px）
- コマ枠は黒線・余白は白・背景色は白
- **ページ番号・ノンブル（数字）は画像内に絶対に描かない**

### セリフ・文字の仕様

- 文字は必ず**縦書き**（上から下・行は右から左）
- セリフに句点（。）は使わない
- 吹き出し内のテキストは**上揃え**
- 吹き出しの背景は**必ず純白（#FFFFFF）**。色・グラデーション・半透明禁止
- フォントイメージ：Noto Sans／アンチックセザンヌ／源英アンチック（明朝系・漫画フォント）
- 文字サイズ（Canva 1000×1414px基準）：通常コマ（3〜4コマ）26pt相当／大ゴマ（1〜2コマ）32〜42pt相当
- モノローグは角丸・薄い吹き出し、ナレーションは四角い枠で区別

### コマ割り・演出技法

- コマの大小を必ず混ぜる（均一な並びは禁止）
- 感情を見せるコマはクローズアップ（顔アップ）
- 世界観・風景は**断ち切り**（コマ枠を端まで広げて開放感）
- 迫力・感情を強めたい所は**ぶち抜き**（キャラが枠を越える・控えめに）
- アクション・緊迫シーンは**斜めコマ**（スピード感）
- **跳ね上げ**＝ページ切り替えで動きを上方向に向ける小技。ぶち抜き・めくりと組み合わせると次ページへの期待感が高まる
- キャラの視線・吹き出しの向きで次のコマへ視線を誘導する
- 静寂・余韻は**間（ま）コマ**（背景だけ）を入れる

### オノマトペ・演出

- オノマトペは手描き文字として絵に溶け込ませる（縦書き・斜め・大小をつけて）
- 効果線・集中線・スピード線を必要に応じて入れる
- 感情マーク（汗・怒りマーク・赤面など）も場面に合わせて添える

### ネガティブプロンプト（標準）

```
lowres, bad anatomy, bad hands, extra fingers, blurry, deformed, watermark, modern objects（時代・設定に合わない現代物）
```

### 生成時の注意

- 同じキャラクターは全コマで顔・髪・服を統一する
- 日本語のセリフはできるだけ正確に描く。吹き出しに収まらないセリフは短く言い換えてよい
- 日本語文字が崩れる場合は吹き出しを空欄にして、後からCanvaで文字を入れる
- **キャラの四面図（キャラシート）を参照画像として必ず添付する**（顔ブレ防止）

### 修正プロンプトのルール

- 直したい点だけを指示し、**画風・スタイルは変えない**と明記する
- **それ以外の部分（コマ割り・構図・他のキャラ）は変えない**と明記する
- キャラ情報・キャラシートを添付し直す
