## 4. 判断基準（迷ったらこの順で考える）

制作でもツール作りでも外注指示でも、迷ったらこの優先順位で決める。

1. **中身が先、見た目は後** — 企画（誰に・何を・どう変わる）を固めてから画像生成に入る。いきなり絵から作らない
2. **初心者にやさしいか** — 専門用語を避け、日本語で丁寧に。むずかしければ却下
3. **クリック・共感されるか** — 表紙は「アートより広告」。漫画は「絵より物語」。読者の心が動くかで判断する
4. **顔ブレしていないか** — キャラクターシートを参照画像として毎回添付し、全ページ同じ顔にする（品質の生命線）
5. **あとで直せる形か** — 表紙のタイトルも漫画本文のセリフも、**基本は画像に入れて一気に生成**する。日本語が崩れたときだけ空欄にしてCanvaで補う。データはバックアップを取る

### タイトル・キャッチの決め方（優先順）

1. 検索キーワードを入れる（「AI漫画」「初心者でも」「〇日で」など、実際に検索される言葉）
2. ベネフィット（読むとどうなるか）を伝える
3. 数字で信頼感を出す（7日・3ステップ 等）
4. 句点（。）は付けない

---

## 5. やらないこと（NG・共通ルール）

- 文字（表紙のタイトル・漫画のセリフ）は**基本は画像に入れて一気に生成**する。日本語が崩れたときだけ空欄にして後からCanvaで補う（表紙・漫画本文共通）
- 企画を固めずにいきなり画像生成を始めない
- 表紙・タイトルに句点（。）を付けない／色数を4色以上にしない
- 作品名に半角アンダースコア（_）を使わない（ツールの保存キーが壊れる）
- 初心者が置いてけぼりになる説明・UIにしない
