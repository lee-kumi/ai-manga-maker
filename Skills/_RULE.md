# Skills フォルダの案内板（_RULE.md）

## このフォルダは何か

**何度も使う「AIへの指示の型（手順書・プロンプトのひな形）」を置く場所。**
毎回いちからお願いを書かなくていいように、うまくいった頼み方を「型」として保存しておく。

## 中に入れるもの

| ファイル | 中身 |
|---|---|
| `〇〇.md` | 1つの作業につき1ファイル。「どんなときに使う」「AIに渡す文面」を書く |

例：`漫画コマ割り.md` / `表紙案づくり.md` / `キャラ設定.md` など、作業ごとに分ける。

## ルール

1. **1スキル＝1ファイル**。ファイル名は作業内容がすぐ分かる日本語で
2. 各ファイルの先頭に「**いつ使うか**（トリガー）」を1行で書く
3. 手順の中身（ルール本文）はここに丸写ししない。**原本（`ナレッジ/` や `docs/`）を参照する薄い作り**にする
4. うまくいった頼み方は、その場で型に足して育てる（使い捨てにしない）

## この事業での実体

このリポジトリには、Claude用の実スキルがすでに `.claude/skills/` にある。

- `/manga-scenario` … 企画書からコマ割りシナリオを作る
- `/manga-prompt` … シナリオから画像生成プロンプトを作る
- `/cover-concept` … 表紙案＋表紙画像プロンプトを作る

いずれも `docs/` のルール文書を原本として参照する形。新しい型を作るときも同じ作りにする。
