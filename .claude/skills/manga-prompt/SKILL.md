# 漫画ページの画像生成プロンプト作成

コマ割りシナリオ（/manga-scenario の出力）を、画像生成AIに貼るプロンプトに変換する。

## 手順

1. まず `docs/manga-maker-rules.md` の「STEP4：画像生成のルール」を読む。**この文書が唯一のルールの原本**であり、本スキルの記述と食い違う場合は文書を優先する
2. 対象ページのシナリオ・画風・キャラ情報を受け取る
3. 文書のルールをすべて織り込んだプロンプトを1ページ分ずつ出力する

## 絶対に守るルール（文書の要点）

- **読む順序は「右上→左上→右下→左下」（日本式・右から左）**を必ず明記する
- **参照画像（キャラクターシート）のキャラ外見を全コマで忠実に再現**させる。ユーザーにはキャラシート添付を毎回念押しする
- キャンバス：1000×1414px（A4縦）・コマ枠は黒線・ガター白10〜15px・ページ番号は描かせない
- セリフは**縦書き**・句点なし・吹き出しは純白・上揃え
- 画風は全ページ共通（勝手に変えない）
- 文書記載の標準ネガティブプロンプトを必ず付ける

## 修正プロンプトを頼まれたら

- 直したい点**だけ**を指示し、「画風・スタイルは変えない」「それ以外のコマ・構図・キャラは変えない」と明記する
- キャラ情報・キャラシートの添付し直しを促す

日本語の文字が崩れる場合は、吹き出しを空欄にして後からCanvaで文字入れする運用を案内する。
