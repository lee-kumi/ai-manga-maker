# コマ割りシナリオ作成

AI漫画メーカーの制作ルールに従って、ページごとのコマ割りシナリオを作る。

## 手順

1. まず `docs/manga-maker-rules.md` を読む。**この文書が唯一のルールの原本**であり、本スキルの記述と食い違う場合は文書を優先する
2. ユーザーから企画情報を受け取る（不足していれば聞く）：
   - テーマ／ペルソナ（たった一人の読者）
   - ストーリー5要素：悩み・本音・壁・カギ・結末
   - 総ページ数・対象の章
3. 文書の「STEP3：コマ割りシナリオのルール」に従い、ページごとに出力する

## 出力形式（1ページ分）

テキスト形式で出力する（表形式 `| --- |` は使わない）：

```
■ Pページ番号（レイアウトパターン：A〜F）
このページで伝えること（1つだけ）：
- コマ1（大きさ・構図）：絵の内容／セリフ「…」／モノローグ（…）／ナレーション［…］
- コマ2 …
めくり（次ページへの引き）：
```

## 絶対に守るルール（文書の要点）

- セリフに句点（。）を使わない・1コマ30文字以内
- セリフ／モノローグ／ナレーションを必ず区別する
- 1ページで伝えることは1つ。クライマックスは大ゴマで止める
- レイアウトパターン（A〜F）を毎ページ変え、同じパターンを3ページ以上続けない
- 各ページの最後は「めくり」＝続きが読みたくなる所で止める

完成したシナリオは `/manga-prompt` で画像生成プロンプトに変換する。
